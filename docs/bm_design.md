## 3. 상세 BM 설계

### 3.1 카드 수집 BM

#### 무료 구조
- 튜토리얼 완료 시 스타터 덱 지급
- 스토리/던전 클리어로 카드 획득
- 일일 무료 뽑기 1회
- SR급 20종 이상 무료 수집 가능

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 일반 뽑기 | 10연차 1회 | 프리미엄 재화 2,500 |
| 픽업 배너 | 특정 카드 확률 UP | 동일 |
| 천장 시스템 | 60연차 레전드리 확정 | - |
| 선택권 패키지 | 10연차 + SSR 선택권 | $14.99 |
| 카드 스킨 | 카드 외형 변경 | $0.99~$2.99 |

### 3.2 덱 편의 BM

#### 무료 구조
- 기본 덱 슬롯 3개
- 덱 추천 시스템 무료
- 카드 도감 무료 열람

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 덱 슬롯 추가 | +3 슬롯 | $1.99 |
| 덱 빌더 프로 | 고급 분석 도구 | $2.99 |
| 카드 복사권 | 같은 카드 추가 획득 | 프리미엄 재화 |
| 빠른 덱 교체 | 대전 중 덱 교체 | 패스 포함 |

### 3.3 전투/성장 BM

#### 무료 구조
- PVE 스토리 무제한
- 일일 PVP 보상
- 카드 강화 재료 파밍

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 강화 패키지 | 카드 강화 재료 대량 | $4.99~$9.99 |
| 레벨 부스트 | 카드 경험치 +50% (7일) | $1.99 |
| 추가 PVP 보상 | 일일 보상 2배 | 광고 or 재화 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 추가 뽑기, PVP 보상, 강화 재료 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $1.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 콘텐츠 무과금 클리어 가능
- 카드 성능보다 덱 구성/시너지가 더 중요
- 시드 덱으로 초보자도 경쟁 가능

### 과금 압박 최소화
- PVE 무제한
- 천장 시스템으로 뽑기 좌절 방지
- 메타 카드 없이도 상위권 진입 가능
