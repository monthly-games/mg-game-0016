## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 0009 카드 풀 계승 |
| 3~6주 | 기본 덱 구조 밸런스 |
| 7~10주 | PVE/PVP 난이도 조정 |
| 11~12주 | 첫 시즌 카드 준비 |

**핵심 과제:**
- 0009 카드 풀 데이터 활용
- 덱 구성 재미 밸런스
- PVE/PVP 밸런스

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 시즌별 카드 추가 1차 |
| 5개월 | 주간 덱 제한 모드 시작 |
| 6개월 | 시즌별 카드 추가 2차 + 밸런스 패치 |
| 7개월 | 신규 히어로 클래스 |
| 8개월 | 시즌별 카드 추가 3차 + 여름 이벤트 |

**핵심 과제:**
- 시즌별 카드 추가 사이클
- 덱 제한 모드 효과 검증
- 히어로 클래스 확장

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 토너먼트 모드 |
| 10개월 | 덱 공유/추천 시스템 |
| 11개월 | 챔피언십 예선 |
| 12개월 | 1주년 챔피언십 본선 |

**핵심 과제:**
- 토너먼트 시스템
- 커뮤니티 기능 강화
- 챔피언십 이벤트
