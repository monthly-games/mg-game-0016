## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 카드 수집/획득                                       │
│         ↓                                               │
│  2. 덱 구축/최적화                                       │
│         ↓                                               │
│  3. 자동 전투 진행                                       │
│         ↓                                               │
│  4. 보상 획득/카드 강화 ─────────────────→ (1로 반복)     │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **카드 수집**: 뽑기, 보상, 제작 등으로 카드 획득
2. **덱 구축**: 카드 조합으로 덱 구성, 시너지 최적화
3. **자동 전투**: 구축한 덱으로 자동 전투 진행
4. **보상/강화**: 전투 보상으로 카드 강화, 새 카드 획득

## 3. 재미 축 (Fun Pillars)

### 3.1 덱 구축 (Deckbuilding)
- **목표**: 나만의 최강 덱을 만드는 전략적 재미
- **핵심 경험**: 이 덱 조합이 정답이야! 하는 발견
- **주요 요소**:
  - 카드 풀 관리
  - 덱 슬롯 제한
  - 코스트 시스템
  - 테마 덱

### 3.2 시너지/콤보 (Synergy)
- **목표**: 카드 간 시너지를 찾아내는 재미
- **핵심 경험**: A+B+C 조합이 대박! 하는 흥분
- **주요 요소**:
  - 카드 시너지 태그
  - 콤보 발동 시스템
  - 세트 효과
  - 숨겨진 시너지

### 3.3 전략 vs 운 (Strategy vs RNG)
- **목표**: 적절한 전략과 운의 균형
- **핵심 경험**: 내 전략이 맞았어! but 운도 따라줬네
- **주요 요소**:
  - 드로우 확률 관리
  - 덱 압축
  - 확률 조작 카드
  - 리스크-리턴 선택

## 4. 디자인 레버 (Design Levers)

### 4.1 카드 중심 전투
| 레버 | 설명 |
|------|------|
| 카드=스킬/패턴 | 전투를 카드 우선 관점으로 재설계 (카드가 곧 스킬/패턴) |
| 구현 방향 | 영웅 기본 공격도 카드화, 스킬 = 카드 사용, 덱 = 전투 행동 풀 |

### 4.2 빌드 다양성
| 레버 | 설명 |
|------|------|
| 풀/레어도/제한 | 카드 풀/레어도/조합 제한으로 빌드 다양성 확보 |
| 구현 방향 | 덱 최대 20장, 레전드리 최대 3장, 같은 카드 최대 2장 |

### 4.3 시드 덱 가이드
| 레버 | 설명 |
|------|------|
| 진입 장벽 완화 | "시드 덱" 예시 제공으로 진입 장벽 완화 |
| 구현 방향 | 초보자용 추천 덱 3종, 카드별 추천 조합, 인기 덱 랭킹 |
