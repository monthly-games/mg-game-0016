## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 카드 손패 하단 배치
- 전투 필드 상단 배치
- 카드 드래그 플레이 최적화

### 3.2 손패 영역

| 요소 | 크기 | 설명 |
|------|------|------|
| 카드 | 80×120dp | 최대 7장 표시 |
| 카드 간격 | 겹침 가능 | 손패 수에 따라 조절 |
| 마나 비용 | 24×24dp | 카드 우측 상단 |

---

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 카드 선택 | 80×120dp | 탭 |
| 카드 플레이 | 손패 → 필드 | 드래그 |
| 카드 상세 | 카드 위 | 롱프레스 |
| 턴 종료 | 버튼 | 탭 |
| 타겟 선택 | 적/아군 | 탭 |

### 터치 피드백
- 카드 선택: 카드 상승
- 플레이 가능: 필드 하이라이트
- 마나 부족: 카드 흔들림 + 빨간색
- 카드 발동: 이펙트 연출
