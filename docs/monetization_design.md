# game_0016 수익화 전략 - 덱빌딩 히어로즈

## 게임 개요
- **장르**: 덱 빌딩 + 자동 전투 + JRPG
- **타깃**: 미드코어, 카드/덱빌딩 선호
- **수익화 우선순위**: IAP > 광고
